H5 自带的表决样式比较丑陋，直接修改 H5 自带的表单样式很难达到上图要求的效果，那么网上的那些漂亮的表单是如何实现的呢？答案是：不用表单。用 CSS 结合 JS 完全可以实现。为什么呢？其他 HTML 元素可以取代表单元素吗？完全可以，其实我们只要理解表单的原理，我们就能用其他的元素代替表单。

### 表单的实质
什么是表单？表单用来干什么？表单是如何运作的？
表单是设计用来统计信息的，然后提交给服务器，由这么几种元素组成：输入框(昵称、密码等)，单选框，复选框，下拉框等，后面三者提供了有限的选择，填好或选择好信息后，可以一键提交到服务器，方便存储。表单在提交时(这里以 GET 请求为例)，实质上是发送了一个 GET 请求，

本页有两种表单控件，一个是日期选择控件，一个是
- 现有模板
- 手动设计
