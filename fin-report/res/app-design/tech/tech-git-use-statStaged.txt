当运行 git add . 命令后，所有文件就进入了暂存区(staging area)，处于暂存状态(staged)，此时所有的文件就会被 git 追踪(盯上了)，任何一丁点细微的修改都会被捕捉到，所以一般查看文件的改动都是在暂存状态查看的。
