## Git 基本原理简介
### Git 如何确保数据的完整性？
修改了任何文件的任何一个字符，新建、删除、重命名、移动了任何一个文件，Git 都会察觉

#### SHA-1 算法
SHA-1 算法用来计算数据的校验和(checksum)，是一个由 40 个十六进制的字符组成的字符串，叫做 SHA-1 哈希值
Git 使用 SHA-1 算法计算数据的哈希值，作为指纹字符串(todo:英文？)
Git 中的文件靠哈希值索引，而不是文件名
